CORE LIBRARY GAP ANALYSIS
=========================

This document identifies functionality from the original Windows application (BiblCal-master/Upgraded) that has NOT yet been migrated to the BiblCalCore library.

ORIGINAL MODULES IN BiblCal-master/Upgraded:
---------------------------------------------
1. ✓ modBiblcalFunctions.cs → CONVERTED to BiblicalCalendarCalculator.cs
   - Core calendar calculation functions
   - Julian day conversions
   - Basic calendar operations

2. ✓ modHebrewCalendarFunctions.cs → CONVERTED to HebrewCalendarFunctions.cs
   - Hebrew calendar calculations
   - Hebrew year types and month names
   - Hebrew leap year calculations

3. ✓ modFlood.cs → CONVERTED to FloodCalculations.cs
   - Flood date calculations (150 days between 17th of 2nd month and 17th of 7th month)
   - Year range analysis for Flood dates
   - Table generation for Flood years

4. ✓ modGolgotha.cs → CONVERTED to GolgothaCalculations.cs
   - Golgotha calculations (Jesus' crucifixion dates)
   - Jordan Crossing calculations
   - Creation date calculations
   - Wave Offering and Pentecost calculations

5. ✓ modSunsets.cs → CONVERTED to SunsetCalculations.cs
   - Sunset time calculations for user locations
   - Year-long sunset tables
   - Location-based sunset calculations

6. ✓ modTimes.cs → CONVERTED to TimesCalculations.cs
   - Sunrise/sunset calculations
   - Moonrise/moonset calculations
   - Moon illumination percentage
   - Daily astronomical time calculations

7. ✓ modDocumentation.cs → CONVERTED to Documentation.cs
   - Help text and documentation strings
   - Module-specific instructions
   - User guidance text

CURRENT BiblCalCore CONTENTS:
------------------------------
- BiblicalCalendarCalculator.cs (basic calendar functions)
